Commands Guide

1. Create a New Virtual Environment

To isolate dependencies for each project, create a new virtual environment:

python3 -m venv env_name

2. Activate the Virtual Environment

Activate the virtual environment for your current shell session:

macOS/Linux:

source env_name/bin/activate

Windows:

.\env_name\Scripts\activate

To deactivate:

deactivate

3. Install Dependencies from requirements.txt

Install dependencies from a requirements.txt file:

pip install -r requirements.txt

4. Create a requirements.txt File

Capture all installed packages for sharing or future use:

pip freeze > requirements.txt

5. Install Common Machine Learning Packages

Quickly install essential libraries:

pip install numpy pandas matplotlib scipy scikit-learn tensorflow torch

6. Running Python Scripts

Run a Python script after activating the environment:

python script_name.py

7. Troubleshooting Dependency Conflicts

Resolve dependency conflicts with these steps:

  1. Check for conflicts:
    2. pip check
  2. Upgrade or downgrade specific packages:
    3. pip install package_name==version

8. Problem-Solving with Common Libraries

Using Scipy's dual_annealing for Optimization:

from scipy.optimize import dual_annealing
# Define and run your dual_annealing function here

Using Pandas for Data Manipulation:

import pandas as pd
df = pd.read_csv('data.csv')
print(df.head())
df['new_col'] = df['col1'] + df['col2']

Visualizing with Matplotlib:

import matplotlib.pyplot as plt
plt.plot([1, 2, 3], [4, 5, 6])
plt.xlabel('X-axis')
plt.ylabel('Y-axis')
plt.title('Sample Plot')
plt.show()

9. Package Management Commands

List Installed Packages:

pip list

Update a Package:

pip install --upgrade package_name

Uninstall a Package:

pip uninstall package_name

10. Clean Up Virtual Environment

Remove the virtual environment if no longer needed:

rm -rf env_name

11. Managing Python Versions

Specify the Python version when creating a virtual environment:

python3.9 -m venv env_name

12. Jupyter Notebook Setup

Use Jupyter notebooks within a virtual environment:

  1. Install Jupyter:
    2. pip install jupyter
  2. Start Jupyter Notebook:
    3. jupyter notebook

13. Add Virtual Environment to Jupyter

  1. Install ipykernel:
    2. pip install ipykernel
  2. Add the environment as a new kernel:
    3. python -m ipykernel install --user --name=env_name --display-name "Python (env_name)"
